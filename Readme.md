# Neovim Config by NJ

## Preqs -

- Language Runtimes
    - NodeJS
    - GCC & G++
    - Python
    - Jdk (Java)
- Other tools
    - Git
    - FZF
    - RipGrep (rg)
    - fd
    - curl
    - make
- Optinal
    - Neovide

## Preqs installation -
**Please check ubuntu-init.sh or fedora-init.sh for installation instruction 
for these operating systems**
